## 문제회고
간단한 문제회고의 경우 retrospect.md 파일을 활용해 파일을 작성하고 있습니다.

회고는 2024-12-18 부터 작성을 시작했고, 미리 풀었던 문제의 경우에도 공부했던 사항이나 고민했던 부분을 남겨볼 예정입니다.

<br>
